# RPGMakerSystems Documentation

## About
RPGMakerSystems is a modular set of classes and logic inspired by RPG Maker, designed for RPGCreator. It provides the foundation for map, event, actor, battle, quest, and UI systems, enabling rapid development of RPGs with familiar workflows.

## Features
- Map & Tile Management
- Event System (triggers, switches, variables)
- Actor/Character System (stats, inventory, equipment, leveling)
- Dialogue & Choices
- Battle System
- Item & Skill Management
- Quest System
- Save/Load System
- UI/UX (menus, HUD, inventory, shops)
- Audio/Visual (music, sound effects, animations)
- Plugin/Script Support
- Database/Editor Integration

## Getting Started
1. Explore the RPGMakerSystems.cs scaffold for class structure.
2. Extend classes with properties and methods as needed for your game.
3. Integrate with the editor UI for visual management of maps, events, actors, etc.
4. Use the plugin/script system for custom logic and extensibility.
